# LUMIÈRE | Elite Reservation Portal 
### *High-Conversion Concierge Interface for Luxury Hospitality*

LUMIÈRE is a professional, bespoke reservation system built for high-end hospitality brands. It prioritizes user trust through minimalist design, responsive logic, and asynchronous data handling.

## 🚀 Live Demo
**Production URL:** [https://gorgeous-quokka-ab3fe7.netlify.app/](https://gorgeous-quokka-ab3fe7.netlify.app/)

---

## 💎 Features & Business Logic
* **Asynchronous Form Submission:** Built with the Fetch API to provide an "app-like" experience with zero page reloads.
* **Smart UI Haptics:** Features visual "shake" feedback for errors and real-time validation to maximize conversion rates.
* **UX/UI Polish:** Includes `autocomplete` attributes for faster mobile filling and ARIA-compliant labels for accessibility.
* **Zero Overhead:** 100% Vanilla JS/CSS—ensuring lightning-fast load times and high PageSpeed scores.
* **Security Conscious:** Front-end validation prevents malformed data injections before API transmission.

## 🛠️ Tech Stack
* **Frontend:** Semantic HTML5, CSS3 (Custom Variables, Flexbox).
* **Logic:** Asynchronous JavaScript (ES6+).
* **Integration:** Formspree API for secure serverless data processing.

## 📂 Installation & Customization
1. Clone the repo: `git clone https://github.com/YOUR-USERNAME/lumiere-system.git`
2. **Security:** Update the `form action` in `index.html` with your unique Formspree Endpoint.
3. **Theming:** Customize brand colors via the `:root` variables in `style.css`.
